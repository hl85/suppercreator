# Markdown to HTML: Themes & Features

Catalog of visual themes and supported Markdown syntax extensions.

## Themes
- **default**: Classic layout, colored H2 backgrounds.
- **grace**: Elegant, rounded cards, text shadows.
- **simple**: Modern minimalist, clean whitespace.
- **modern**: Large radius, pill-shaped titles.

## Supported Features
- Code blocks with highlighting, Math, PlantUML/Mermaid, Footnotes, Alerts, Ruby text.
